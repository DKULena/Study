Truthy & Falsy
True/False로 '취급'되는 값
Truthy - Falsy 빼고 전부
Falsy(7개) - undefined, null, 0, -0, NaN, ""(빈문자열), 0n
단락평가(Short-circuit Evaluation)
논리 연산자 사용 시 평가 최적화
AND 연산자 (&&) - 처음 만나는 False값 반환, 둘 다 True면 두번째 값 반환
OR 연산자 ( || ) - 첫 번째 값이 true면 첫 번째 값을 반환, false면 두 번째 값을 반환
구조분해할당(destructuring)
배열이나 객체의 속성을 분해해서 값을 개별 변수에 담을 수 있게 하는 JavaScript 표현식

배열 구조분해할당

let arr = [1, 2, 3];

let [one, two, three] = arr;

Result = one:1, two:2, three:3

객체 구조분해할당

let person = {
name: "이성재",
age: 27,
hobby: 코딩
}

let {name, age, hobby} = person;

Result = name:이성재, age:27, hobby:코딩

객체 구조분해할당을 이용해서 함수의 매개변수를 받기

const func = ({ name, age, hobby }) {
  console.log(name, age, hobby)
}

Result = name:이성재, age:27, hobby:코딩

원시타입 VS. 객체타입
두 타입으로 나눠놓는 이유는 값이 저장되거나 복사되는 과정이 다르기 때문
원시타입: Number, String, Boolean, Null, Undefined
객체타입: Array, Object, RegexExp(정규표현식)
