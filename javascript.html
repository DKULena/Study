Truthy & Falsy
True/False로 '취급'되는 값
Truthy - Falsy 빼고 전부
Falsy(7개) - undefined, null, 0, -0, NaN, ""(빈문자열), 0n
✨단락평가(Short-circuit Evaluation)
논리 연산자 사용 시 평가 최적화
AND 연산자 (&&) - 처음 만나는 False값 반환, 둘 다 True면 두번째 값 반환
OR 연산자 ( || ) - 첫 번째 값이 true면 첫 번째 값을 반환, false면 두 번째 값을 반환
구조분해할당(destructuring)
배열이나 객체의 속성을 분해해서 값을 개별 변수에 담을 수 있게 하는 JavaScript 표현식

배열 구조분해할당

let arr = [1, 2, 3];

let [one, two, three] = arr;

Result = one:1, two:2, three:3

객체 구조분해할당 

let person = {
name: "이성재",
age: 27,
hobby: 코딩
}

let {name, age, hobby} = person;

Result = name:이성재, age:27, hobby:코딩

객체 구조분해할당을 이용해서 함수의 매개변수를 받기 

const func = ({ name, age, hobby }) {
  console.log(name, age, hobby)
}

Result = name:이성재, age:27, hobby:코딩

원시타입 VS. 객체타입
두 타입으로 나눠놓는 이유는 값이 저장되거나 복사되는 과정이 다르기 때문 (얕은복사 vs. 깊은복사)
원시타입(불변값): Number, String, Boolean, Null, Undefined

값 자체로써 변수에 저장, 복사

객체타입: Array, Object, RegexExp(정규표현식)

참조값(주소값)을 통해 변수에 저장, 복사

배열 메서드(1)
요소조작: push(), pop(), shift(), unshift(), slice(), concat()
let arr = [1, 2, 3];
1. push() : 배열의 맨 뒤에 새로운 요소를 추가하는 메서드

arr.push(4);

arr = [1, 2, 3, 4]

2. pop() : 배열의 맨 뒤에 새로운 요소를 제거하고 반환

arr.pop();

arr = [1, 2]

3. shift() : 배열의 맨 앞에 있는 요소를 제거, 반환

arr.shift()

arr = [2]

4. unshift() : 배열의 맨 앞에 새로운 요소를 추가

arr.shift(0)

arr = [0, 2]

5. slice() : 가위처럼 배열의 특정 범위를 잘라서 새로운 배열로 반환
(기존 배열이 바뀌지는 않음)

arr = [1, 2, 3, 4, 5]

let slicedArr = arr.slice(2, 5) {2 <= arr < 5}

slicedArr = [1, 2]

6. concat() : 두개의 서로 다른 배열을 이어 붙여서 새로운 배열을 반환

let arr1 = [1, 2]

let arr2 = [3, 4]

concatedArr = arr1.concat(arr2)

concatedArr = [1, 2, 3, 4]
